Hrt-for-Vela 项目

快速上手

1. 开发

开发命令
npm install
npm run start

执行以上命令安装依赖并启动开发服务器。

2. 构建

构建命令
npm run build
npm run release

执行构建命令生成生产环境代码，然后发布到快应用平台。

3. 调试

调试命令
npm run watch

启动监视模式，可以实时查看代码修改效果。

4. 代码规范化配置

代码规范化可以帮助开发者在git commit前进行代码校验、格式化、commit信息校验。

使用前提：必须先关联git仓库

macOS 或 Linux 系统：

sh husky.sh

Windows 系统：

./husky.sh

了解更多

你可以通过我们的官方文档熟悉和了解快应用。
